## 🇬🇧 English

**Archive Scanner** is a fast, secure, and modern open-source tool designed to help you manage your network configurations and scans efficiently. Built with performance and user experience in mind, it provides a seamless interface for your daily network tasks.

### Key Features
* ✨ **Material 3 UI:** A sleek, intuitive, and modern interface built with the latest design standards.
* ⚡ **High Performance:** Optimized scanning engine to deliver results faster.
* 🛡 **Privacy First:** Open-source and developed by **Bachelor⚡️** with a focus on your security.
* 📱 **Full Compatibility:** Works smoothly on modern Android versions.
