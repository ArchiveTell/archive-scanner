# 🚀 Archive Scanner

## 🇬🇧 English

Archive Scanner is a powerful, open-source Android network utility designed to scan for clean Cloudflare Edge IPs and seamlessly generate proxy configurations. Built with performance and user experience in mind, it eliminates the hassle of manual address swapping and streamlines your network bypass setup.

### Key Features
* 🔍 **Smart IP Scanner:** Quickly scan and find healthy, low-latency Cloudflare edge IPs.
* 🛠️ **Auto Config Generator:** Input a base VLESS or Trojan config, and instantly generate ready-to-use configs for every clean IP found. It intelligently swaps the connection address while strictly preserving your SNI, port, and path.
* 📋 **One-Tap Export:** Copy all generated configs (one per line) directly to your clipboard with a single tap. Config names are automatically tagged with their respective IPs for easy identification.
* ✨ **Material 3 UI:** A sleek, intuitive, and modern interface built with the latest Android design standards.
* 🛡️ **Privacy First:** Fully open-source and developed by Bachelor⚡️. All scans and generations happen locally on your device.

## 🇨🇳 中文 (Chinese)

Archive Scanner 是一款强大且开源的 Android 网络实用工具，专为扫描干净的 Cloudflare 边缘 IP 并无缝生成代理配置而设计。我们注重性能与用户体验，为您省去手动替换地址的繁琐步骤，简化您的网络配置流程。

### 主要功能
* 🔍 **智能 IP 扫描:** 快速扫描并查找健康、低延迟的 Cloudflare 边缘（优选）IP。
* 🛠️ **配置自动生成:** 输入基础的 VLESS 或 Trojan 节点链接，自动为每个干净 IP 生成可用的节点。智能替换连接地址，同时严格保留您的 SNI、端口和路径设置。
* 📋 **一键导出:** 只需点击一次，即可将所有生成的配置（每行一个）复制到剪贴板。配置名称会自动添加对应的 IP，方便在客户端中识别。
* ✨ **Material 3 界面:** 采用最新的 Android 设计标准，界面简洁直观，交互流畅。
* 🛡️ **隐私至上:** 完全开源，由 Bachelor⚡️ 开发。所有的扫描和生成都在您的设备本地完成。
